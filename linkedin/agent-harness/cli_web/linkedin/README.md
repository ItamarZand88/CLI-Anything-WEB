# cli-web-linkedin

> Generated by [CLI-Anything-Web](../../../../cli-anything-web-plugin/) from [linkedin.com](https://www.linkedin.com)

Search LinkedIn for people, jobs, and companies. View profiles, feed, and company pages. Post, edit, delete updates. React, unreact, comment, edit and delete comments. View notifications, manage network connections, and send messages. Requires authentication — uses browser login to capture session cookies.

24 commands across 10 groups: auth, feed, search, profile, company, jobs, post, notifications, network, messaging.

## Installation

```bash
pip install -e linkedin/agent-harness
```

## Authentication

```bash
cli-web-linkedin auth login     # Opens browser for LinkedIn login
cli-web-linkedin auth status    # Check if authenticated
cli-web-linkedin auth logout    # Remove saved auth
```

## Usage

### Feed

```bash
cli-web-linkedin feed --count 10
cli-web-linkedin feed --json
```

### Search

```bash
cli-web-linkedin search all "python developer" --limit 10
cli-web-linkedin search people "python developer" --limit 10
cli-web-linkedin search companies "AI startup" --json
cli-web-linkedin search posts "hiring" --json
```

### Profiles

```bash
cli-web-linkedin profile me --json
cli-web-linkedin profile get williamhgates --json
```

### Companies

```bash
cli-web-linkedin company anthropic --json
```

### Jobs

```bash
cli-web-linkedin jobs search "data scientist" --limit 10 --json
cli-web-linkedin jobs get JOB_ID --json
```

### Posts, Reactions & Comments

```bash
cli-web-linkedin post create "Hello LinkedIn!"
cli-web-linkedin post edit "urn:li:activity:123" "Updated text"
cli-web-linkedin post delete "urn:li:activity:123"
cli-web-linkedin post react "urn:li:activity:123" --type LIKE
cli-web-linkedin post unreact "urn:li:activity:123"
cli-web-linkedin post comment "urn:li:activity:123" "Great post!"
cli-web-linkedin post edit-comment "urn:li:comment:456" "Edited comment"
cli-web-linkedin post delete-comment "urn:li:comment:456"
```

### Notifications

```bash
cli-web-linkedin notifications --limit 10 --json
```

### Network

```bash
cli-web-linkedin network connections --json
cli-web-linkedin network invitations --json
cli-web-linkedin network connect "urn:li:fsd_profile:ABC" -m "Let's connect!" --json
```

### Messaging

```bash
cli-web-linkedin messaging list --json
cli-web-linkedin messaging send "urn:li:fsd_conversation:123" "Hello!" --json
```

## Protocol

- **Website**: https://www.linkedin.com
- **Protocol**: GraphQL + Voyager REST API hybrid
- **Auth**: Cookie-based (li_at + JSESSIONID), browser login via Playwright
- **HTTP library**: curl_cffi (PerimeterX bypass)
- **Search**: Headless Playwright for people/companies/posts (RSC-based pages)
