# Examples

Real CLIs generated by CLI-Anything-Web:

| Example | App | Architecture | Auth |
|---------|-----|-------------|------|
| [**futbin**](futbin/) | EA FC 26 database | Server-rendered HTML + JSON search API | Cookie-based (optional) |

## Try it

```bash
cd examples/futbin
pip install -e .
cli-web-futbin --help
```

Each generated CLI includes:
- Full Python CLI with Click commands and REPL mode
- Auth management (browser login, cookie import)
- `--json` output for agent consumption
- Unit and E2E tests
- Software-specific SOP document (`<APP>.md`)

## Structure

Every generated CLI follows the same layout:

```
<app>/
├── <APP>.md              # API surface documentation
├── setup.py              # Package config
└── cli_web/<app>/
    ├── README.md         # Usage guide
    ├── <app>_cli.py      # Main CLI entry point
    ├── core/             # HTTP client, auth, session, models
    ├── commands/         # Click command groups
    ├── utils/            # REPL skin, output formatters
    └── tests/            # Unit + E2E tests
```
