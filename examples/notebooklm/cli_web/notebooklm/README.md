# cli-web-notebooklm

> Generated by [CLI-Anything-Web](../../../../cli-anything-web-plugin/) from [notebooklm.google.com](https://notebooklm.google.com)

CLI for Google NotebookLM — AI-powered research and note-taking. Manage notebooks, sources, chat, and studio artifacts (audio overviews, presentations, quizzes).

## Installation

```bash
cd examples/notebooklm
pip install -e .

# For browser-based login:
pip install -e ".[browser]"
```

## Usage

```bash
# List notebooks
cli-web-notebooklm notebooks list

# Get notebook details
cli-web-notebooklm notebooks get --id <notebook-id>

# List sources
cli-web-notebooklm sources list --notebook-id <id>

# Chat history
cli-web-notebooklm chat history --notebook-id <id>

# Suggested questions
cli-web-notebooklm chat suggested --notebook-id <id>

# List artifacts (audio, video, presentations)
cli-web-notebooklm artifacts list --notebook-id <id>

# REPL mode
cli-web-notebooklm
```

## Auth

NotebookLM requires Google authentication.

```bash
cli-web-notebooklm auth login                              # Playwright browser login
cli-web-notebooklm auth login --from-browser               # Extract from Chrome debug session
cli-web-notebooklm auth login --cookies-json cookies.json  # Import from file
cli-web-notebooklm auth status                             # Check auth status
```

## JSON Output

Add `--json` before any command for structured output:

```bash
cli-web-notebooklm --json notebooks list
```
