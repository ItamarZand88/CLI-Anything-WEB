# cli-web-suno

> Generated by [CLI-Anything-Web](../../../../cli-anything-web-plugin/) from [suno.com](https://suno.com)

CLI for Suno — AI music generation platform. Generate songs from text prompts, manage your library, organize projects, and explore trending content.

## Installation

```bash
cd examples/suno
pip install -e .

# For browser-based login:
pip install -e ".[browser]"
```

## Usage

```bash
# List your songs
cli-web-suno songs list

# Generate a song
cli-web-suno songs generate --description "a happy pop song about coding"

# Download a song
cli-web-suno songs download --id <song-id> --output song.mp3

# Check generation status
cli-web-suno songs status

# View billing/credits
cli-web-suno billing info

# Explore tags
cli-web-suno explore tags

# List projects
cli-web-suno projects list

# REPL mode
cli-web-suno
```

## Auth

Suno requires authentication for all operations.

```bash
cli-web-suno auth login                              # Playwright browser login
cli-web-suno auth login --from-browser               # Extract from Chrome debug session
cli-web-suno auth login --cookies-json cookies.json  # Import from file
cli-web-suno auth status                             # Check auth status
```

## JSON Output

Add `--json` before any command for structured output:

```bash
cli-web-suno --json songs list --limit 5
```
